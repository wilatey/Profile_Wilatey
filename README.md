THis is the self project for designing own profile by react

## Installation
- react@19.1.0
- tailwindcss@4.1.11
- vitejs/plugin-react@4.6.0
- tailwindcss/vite@4.1.11
- react-router-dom@7.6.3

## Features
- Responsive UI
- Customized animation by tailwindcss
- Toggle dark and light mode
  - fancy background animation in dark mode
  - coloring with user-friendly in light mode

## Configure
`npm run dev`
It is for running by npm with window popping
